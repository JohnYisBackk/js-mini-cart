# 🛒 Mini Cart 

A modern mini e-commerce cart application built with **HTML, CSS and Vanilla JavaScript**.

This project demonstrates a fully functional shopping cart system with dynamic UI updates, local storage persistence, and a clean premium dark design.

---


🚀 **Live Demo:** https://johnyisbackk.github.io/js-mini-cart/


---

## 🚀 Features

- 🧾 Product listing (dynamic rendering)
- ➕ Add to cart
- ➖ Increase / decrease quantity
- ❌ Remove items from cart
- 💰 Total price calculation
- 🔢 Cart item counter (total quantity)
- 💾 LocalStorage persistence
- 🧠 Event delegation (clean JS architecture)
- 🛍️ Checkout modal (demo flow)
- 🧹 Cart reset after order confirmation

---

## 🎯 Tech Stack

- HTML5
- CSS3 (custom properties, dark UI)
- Vanilla JavaScript (ES6)

---

## 🧠 Key Concepts

- DOM manipulation
- State management (cart array)
- Array methods (`map`, `find`, `filter`, `reduce`)
- Event delegation
- LocalStorage API

---

## ⚙️ How It Works

1. Products are rendered dynamically from a JS array  
2. User can add/remove/update items in the cart  
3. Cart state is stored in localStorage  
4. UI updates automatically after every action  
5. Checkout triggers a modal and resets the cart
